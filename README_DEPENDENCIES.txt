# Dependency Installation Instructions

To run this project, you must install the required Python packages. There are two ways to do this:

## 1. Using requirements.txt (Recommended)

1. Open a terminal (Command Prompt or PowerShell) in this folder.
2. Run:

    pip install -r requirements.txt

This will install all required dependencies, including pandas, numpy, matplotlib, seaborn, PyQt5, and pyperclip.

## 2. Using the Auto-Installer Script

If you prefer, you can run the provided script to automatically install any missing dependencies:

    python install_requirements.py

This script will check for missing packages and install them as needed.

---

If you encounter errors about missing packages, ensure you are using the correct Python environment and have internet access.
